# Quran - [Expressjs]

## Installation
To get the Node server running locally:
```
git clone https://github.com/luthfikamaal/quran-expressjs.git
cd quran-expressjs
npm install
npm start
```
Run ```localhost:8000``` in web browser.

## Tools
* [Node.js](https://nodejs.org/)
* [Git](https://git-scm.com/)

## Data source
